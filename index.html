1 HTML Structure

<!DOCTYPE html> <html> <head> <title>Page Title</title> </head> <body> Content goes here </body> </html>

Parts:

  • <!DOCTYPE html> ‚Üí Declares HTML5
  • <html> ‚Üí Root element
  • <head> ‚Üí Contains page info (not visible)
  • <body> ‚Üí Contains visible content

<h1>Main Heading (Biggest)</h1> <h2>Sub Heading</h2> <h3>Smaller Heading</h3> <h4>Small Heading</h4> <h5>Smaller</h5> <h6>Smallest</h6>

3 Paragraphs & Text

<p>This is a paragraph.</p> <p>This is another paragraph.</p> <span>Inline text</span> <span>More inline text</span> <br> ‚Üê Line break <hr> ‚Üê Horizontal line

This is a paragraph.

This is another paragraph.

Inline text More inline text

Difference:

  • <p> ‚Üí Block element (starts new line)
  • <span> ‚Üí Inline element (stays in same line)

4 Links (Anchor Tag)

<a href="https://google.com">Visit Google</a> <a href="page.html" target="_blank">Open in New Tab</a> <a href="#section">Jump to Section</a>
Example Link

Attributes:

  • href ‚Üí Where link goes (URL or file)
  • target="_blank" ‚Üí Opens in new tab

9 Semantic Tags

<header>Top section</header> <nav>Navigation</nav> <section>Content section</section> <article>Blog post</article> <footer>Bottom section</footer>

Why use them?

  • Better for SEO
  • Easier to read code
  • Better accessibility
Tip: Use semantic tags instead of <div> when possible.
